# Template Overview

This section provides report templates for different development boards and examples, aiming to help contributors document and share their test results. Each template includes the necessary structure and format to ensure clarity and consistency of information.

## Usage Instructions
1.**Development Board Documentation Template**: Located at `templates/[board-name]/README.md`, used to record basic information and hardware specifications of a specific development board.

2.**Example Documentation Template**: Located at `templates/[board-name]/[os-name]/README.md`, used to record the build and runtime environment, execution steps, and actual results of a specific example.

Please fill in and modify the corresponding template content according to the specific development board and example.
